# شُبّاك كوفي | Shubbak Coffee ☕

> قهوة مختصة «على السريع» بنكهات موزونة ونوتات واضحة

موقع إلكتروني تسويقي لكوفي شوب **شُبّاك** — قبل الافتتاح.

📍 القصيري · داون تاون أم الحمام · القطيف

---

## التقنيات

- **React 18** + **Vite**
- CSS-in-JS (inline styles)
- خطوط عربية: Aref Ruqaa, Reem Kufi, Amiri, Tajawal
- أنيميشن CSS + Intersection Observer
- تصميم متجاوب (Mobile-first)

## التشغيل المحلي

```bash
npm install
npm run dev
```

## البناء للنشر

```bash
npm run build
```

الملفات الجاهزة للنشر ستكون في مجلد `dist/`

## النشر

يمكن نشر المشروع على:
- **Vercel** — `vercel deploy`
- **Netlify** — اربط الريبو مباشرة
- **GitHub Pages** — مع إعداد `base` في `vite.config.js`
